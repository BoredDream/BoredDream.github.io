github+jsdelivr+PicGo搭建图床

发表于 2025-08-23 更新于 2026-01-21 分类于 博客搭建及diy 阅读时长 ≈ 1 分钟

一、获取github token（给予picgo上传权限）
1.创建github仓库BlogIMG

2.进入主页右上角->setting->developer settings->personal token(classic)->Creat new tokens(classic),保存好只显示一次
二、配置PicGo
1.下载对应版本PicGo

2.进入点击侧边栏图床设置，选择github，新建，图床配置名自定义，仓库名github名字/token对应仓库名，token,路径自定义，自定义域名https://cdn.jsdelivr.net/gh/自己github名字/仓库名
3.可以开始用了！直接在picgo上传图片，得到的链接直接插入博客markdown里面
